1. 音效
	1.1. 更新公告
		B站视频发布时有81个音频文件，现已更新至277个。
		使用新版音频文件包前请先删除旧版音频文件。
	1.2. 使用方法
		您可使用 reg 文件导入声音方案，然后任务栏音量图标右键→声音（Win11：声音设置→更多声音设置），
		声音方案选择2（1是 UP 主自用的，包含一些其他的音效）。
		安全提示：导入前，可先用文本编辑器打开它来检查代码。
	1.3. 我懒得去一个个设定音效。
		UP 主自用声音方案在视频简介→百度网盘链接→原神游戏音效选集→相关附件→配置文件→此文件夹下的 reg 文件。
		下载并打开之，系统会提示导入注册表，导入后打开「声音」控制面板切换声音方案即可。
	1.4. 想要中配（或英配韩配）派蒙的音量调节语音。
		很抱歉，UP 主实在不情愿为了录制几秒钟音频而去下载原神那硕大的语音包。
		您可以将游戏音量调至 10 0 4 0 左右，然后自行内录 wav 文件。
		音频内录推荐使用 OBS Studio：https://github.com/obsproject/obs-studio

2. 鼠标指针
	2.1. 更新公告
		「CZJSTMC」现已更名为「STMC」，即「Sam Toki's Mouse Cursors」的首字母。
		2023/08/06 大型更新：
			此次更新为大型更新。更新后的作品与以前视频演示中的鼠标指针差异较大。
			改进了很多，也废除了很多质量欠佳的鼠标指针，且更改了安装配置。
			若您使用过旧版本的鼠标指针，请（参考 PDF 教程）卸载后再重装。
			若想要使用一些已被废除的旧版鼠标指针，请下载旧版存档（Archive 文件包）。
	2.2. 使用
		2.2.1. 使用方法
			请查看「使用教程」，或进入「STMC」文件夹查看「INF 文件使用说明」。
		2.2.2. Win11 能用吗？
			从 WinXP 到 Win11 均可使用。
		2.2.3. 右键菜单没有「安装」。
			① 请确保您已经解压了压缩包，然后再操作 inf 文件。
			② Win11 系统需要先点击「显示更多选项」调出经典右键菜单。
				可以通过修改注册表让 Win11 默认使用经典右键菜单，具体方法请自行搜索。
			③ 请留意您是否单击选中了 inf 文件然后再点右键。
				据我观察他人使用电脑的经验，很多人忘记先左键选中文件然后再右键，导致他们相当于在空白处调出右键菜单。
				这种右键菜单是针对当前文件夹，而不是您想选中的文件，自然不会出现诸如安装 inf 的选项。
		2.2.4. 如何卸载？
			运行「control mouse」打开「鼠标 属性」窗口，点击「指针」选项卡，逐个删除所有「STMCS」方案。
			然后打开资源管理器，删除 C:\Windows\Cursors\STMC
	2.3. 特定鼠标指针
		2.3.1. 看不到纳西妲长按E技能绿色方框鼠标指针。
			「精确选择」鼠标指针会在一些图像编辑软件与屏幕截图软件中出现，比如试试按下 Win+Shift+S。
			如果很喜欢这个方框，可以自行把它设为「正常选择」，不过可能会有些违和。
	2.4. 关于设计理念
		2.4.1. 什么是「概念鼠标指针」？
			「概念鼠标指针」不同于将人物头部形象绘制于鼠标指针的设计风格，其利用了人物形象中某些具备代表性的几何图形特征。
			以极简主义的设计理念，在保证代表性与美观性的前提下，兼顾鼠标指针的实用性。
		2.4.2. 想要带有Q版人物形象的鼠标指针。
			我不擅长这种设计。若您偏好这种设计，不妨找找其他作者的作品。

3. 字体替换
	3.1. MacType 的风险
		MacType 软件功能涉及 Windows 系统底层，因此具有一定的不稳定性，但风险程度相对较低，可以放心。
		不过切勿在新版本 Windows 上使用过旧版本的 MacType。从 Win7 至稳定版的 Win11，搭配最新版本的 MacType 基本没问题。
		万一安装使用 MacType 导致系统崩溃，可进入 Windows 安全模式然后禁用或卸载 MacType。
	3.2. 使用建议
		游戏 UI 使用的字体，比如原神的「汉仪文黑-85W」，其实并不适合日常用作系统 UI 字体来显示大量小号文字。
		建议尝尝鲜即可，用腻了就换回去。推荐将其用作桌面歌词字体，效果挺不错。
	3.3. 关于 noMeiryoUI 软件
		MacType 可搭配 noMeiryoUI 使用。非必要，但效果更佳。
		noMeiryoUI 是一款更换 Windows 系统 Win32 程序 UI 字体的小工具。由于仅对 Win32 有效，因此具有局限性。
		https://github.com/Tatsu-syo/noMeiryoUI
